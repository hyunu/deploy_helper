# 폰트 다운로드 가이드

## Noto Sans KR 폰트 다운로드

1. Google Fonts에서 다운로드:
   - https://fonts.google.com/noto/specimen/Noto+Sans+KR
   - "Download family" 버튼 클릭
   - 압축 해제 후 다음 파일들을 이 폴더에 복사:
     - `NotoSansKR-Light.woff2` (또는 .woff)
     - `NotoSansKR-Regular.woff2` (또는 .woff)
     - `NotoSansKR-Medium.woff2` (또는 .woff)
     - `NotoSansKR-Bold.woff2` (또는 .woff)

2. 또는 명령어로 다운로드 (macOS/Linux):
```bash
# 이 폴더에서 실행
curl -L "https://fonts.google.com/download?family=Noto%20Sans%20KR" -o noto-sans-kr.zip
unzip noto-sans-kr.zip
# 필요한 파일만 이 폴더로 이동
```

## 파일 구조
```
public/fonts/
├── NotoSansKR-Light.woff2
├── NotoSansKR-Regular.woff2
├── NotoSansKR-Medium.woff2
├── NotoSansKR-Bold.woff2
└── README.md
```
