# 폰트 파일 저장 위치

이 폴더에 폰트 파일을 저장하세요.

## 폰트 파일 형식
- `.woff2` (권장)
- `.woff`
- `.ttf`

## 폰트 파일 이름 예시
- `NotoSansKR-Regular.woff2`
- `NotoSansKR-Bold.woff2`
- `LGSmart-Regular.woff2` (LG Smart 폰트를 사용하는 경우)

## 사용 방법
1. 폰트 파일을 이 폴더에 저장
2. `src/index.css`에서 `@font-face`로 폰트 정의
3. CSS에서 `font-family`로 사용

## 폰트 다운로드
- Google Fonts: https://fonts.google.com/noto/specimen/Noto+Sans+KR
- LG Smart 폰트: 공식 라이선스 확인 후 사용
